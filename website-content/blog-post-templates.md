## SEO CHECKLIST FOR EVERY POST

Before publishing, ensure:

- [ ] **Title:** Includes primary keyword, under 60 characters
- [ ] **Meta description:** 150-160 characters, includes keyword
- [ ] **URL slug:** Short, keyword-rich, no dates
- [ ] **H1:** Only one, matches title
- [ ] **H2s:** Include secondary keywords
- [ ] **Images:** Alt text with keywords, compressed
- [ ] **Internal links:** 2-3 to other blog posts
- [ ] **External links:** 1-2 to authoritative sources
- [ ] **CTA:** Clear call-to-action (booking, lead magnet)
- [ ] **Pinterest image:** Vertical (2:3), text overlay, saved in post
- [ ] **Categories/tags:** Consistent with content pillars

## BATCH WRITING WORKFLOW

### Day 1: Research & Outline (All Posts)
- Keyword research for 4 posts
- Create outlines using templates
- Gather reference images

### Day 2: Write First Drafts
- Write 2 posts using templates
- Don't edit - just write

### Day 3: Write + First Edit
- Write remaining 2 posts
- Light edit on Day 2 posts

### Day 4: Final Edit & Publish Prep
- Final edits on all posts
- Create Pinterest images
- Schedule posts
- Set up email sequences
