## FILMING CHECKLIST

### Before Each Short:
- [ ] Phone charged (100% for batch filming)
- [ ] Ring light or natural light ready
- [ ] Background clean or intentionally messy (for before shots)
- [ ] Props/products gathered
- [ ] Script printed or on tablet nearby
- [ ] Multiple outfit options for variety

### During Filming:
- [ ] Film vertical (9:16)
- [ ] Multiple takes of each section
- [ ] Film extra B-roll for flexibility
- [ ] Get "before" shots BEFORE organizing
- [ ] Natural expressions > perfect delivery

### After Filming:
- [ ] Import to editing app (CapCut recommended)
- [ ] Add captions (80% watch without sound)
- [ ] Add trending audio if applicable
- [ ] Export at highest quality
- [ ] Write caption and hashtags before posting
